# Projet Unity 3D

## Description

C'est un jeu a la 3e personne, dans un monde dont le visuel rappelle une prairie et dont le but est de capturer
le plus d'animaux possible avant que le compteur ne se termine.

## Inputs

WASD/Fleches directionelles pour se deplacer
ESPACE pour sauter
SHIFT pour courir
E pour interagir avec les animaux
